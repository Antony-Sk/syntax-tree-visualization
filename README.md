# Syntax tree visualizer

Web application for visualizing abstract syntax tree and syntax array of the code

## How to build
First, install npm
Mac:
```shell
brew install node
```
Ubuntu:
```shell
sudo apt install nodejs
```
Arch:
```shell
pacman -Suy nodejs npm
```

Windows sucks

And the yarn
```shell
npm install --global yarn
```

Then start the server
```shell
yarn install 
yarn start
 ```
</br>Server started
